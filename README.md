# MangaDex Desktop Client
Great manga reader made with best coding language RUST.

---

## Install

1. Needs Rust and GTK4/WebKit:
   `sudo apt install librust-gtk4-dev libwebkit6-gtk-6.0-dev` (Ubu/Deb)
   `sudo pacman -S gtk4 webkit6gtk` (Arch)

2. Install Script:
   ```bash
   chmod +x install.sh
   ./install.sh
